# Final-Project-Exam-DBMS-for-Sustainable-Development
This project is about a school database system using SQL to manage records of student, courses, enrollment,  and frades of students, the goal of our project is to make storing and accessing information easier and more organize compare to a manual record keeping
There database is designed to avoid any duplication of data and errors by using proper table relationship. It also follow normalization rule so that the data stays clean and consistent.
Overall,  this project shows how a database can help school manage records more efficiently.
